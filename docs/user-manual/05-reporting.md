1. Navigate to the report
2. Click **Export** button
3. Select format
4. Choose data range
5. Download file

Set up automatic report delivery:

1. Go to **Settings** â†’ **Scheduled Reports**
2. Click **+ New Schedule**

   - Report type
   - Frequency (daily, weekly, monthly)
   - Recipients (email addresses)

## Troubleshooting

### Data Not Updating

- Reports cache for performance
- Click **Refresh** to get latest data
- Some reports update hourly

### Missing Data

- Check date range filter
- Verify permissions to see data
- Data might be filtered out

### Export Fails

- Try smaller date range
- Check file size limits
- Try different format
